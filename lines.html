Let us sample some points from a line $L:y=mx+c$ and attempt to fit a line $L_1:y=m_1x+c_1$ to the data points. Using an iterative algorithm like stochastic gradient descent guarantees convergence of the parameters $m_1, c_1$ to the true slope and intercept, $m, c.$ We don't add any noise anywhere and this is as simple as things get when one starts out in deep learning.
