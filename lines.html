Linear Regression with Multiple Lines

Looking at the convergence of parameters of individual lines when multiple lines are used to perform linear regression.

We cannot approximate a non-linear function using a neural network unless we use non-linear activation functions. This made me wonder if we could approximate a linear function by injecting non-linearity into a network. I was using a very primitive network that learnt two lines with a ReLU non-linearity added to the output of the first line. Those results aside, I found something interesting when visualizing the weights (by plotting the two lines). Both lines always had the same slope, or at least the same magnitude. Which leads us to the following analysis.

I have to mention that I searched far and wide for what this exact setting is called, and I couldn't find a name for it. This is probably because the problem is very trivial. But I wish it were trivial to name this blog post, because the one I have right now is very underwhelming, maybe misleading, and probably confusing.

Problem Statement

Sample a set of points $\{(x_1, y_1),(x_2, y_2),\dots, (x_n, y_n)\}$ from the line $L:y=mx+c$ and fit two lines $L_1,L_2$ such that $L=L_2(L_1(x))$ where $L_1:y=m_1x+c_1$ and $L_2:y=m_2x+c_2.$ At convergence, we get $$L=m_1m_2x+m_2c_1+c_2,$$ and $$m=m_1m_2\qquad c=m_2c_1+c_2.$$ From choosing a few examples for the original line, we can quickly see that there are infinitely many solutions for the two lines to take (try $L:y=x$ and see that all pairs of lines $L_1:y=ax,$ and $L_2:y=x/a$ are solutions, $a\in\mathbb{R}-\{0\}$)). I now have some questions,

Question 1: On using gradient descent to solve for the two lines, $m_1$ and $m_2$ always converge at $\pm\sqrt{m}.$ Can this be proven analytically? Why does this happen when there are infinitely many solutions.

Question 2: Is this behaviour general? If we use $n$ lines, will the slopes of all the lines converge to $\pm (m)^{1/n}$?

Question 3: What can we say about the intercepts ($c_1, c_2$)? They don't converge to a single number like the slopes do, so maybe we don't care that much.
